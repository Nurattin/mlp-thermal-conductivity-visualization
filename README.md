# MLP Forward Animation

Интерактивная HTML-визуализация прямого прохода MLP-модели для прогноза коэффициента теплопроводности пород.

Визуализация показывает:

- входные признаки петрофизического датасета;
- слои MLP: `4 -> 128 -> 64 -> 32 -> 1`;
- веса связей, Batch Normalization, ReLU и Dropout;
- пошаговый прямой проход;
- режим обучения с графиком функции потерь;
- формулы прямого и обратного прохода.

## Структура

```text
.
├── index.html       # основная интерактивная визуализация
├── mlp_dataset.js   # встроенный датасет для демонстрации
├── package.json     # удобный локальный запуск через npm
└── README.md
```

## Быстрый запуск

Откройте файл `index.html` в браузере.

Если браузер блокирует локальные скрипты, запустите простой локальный сервер.

### Через Python

```bash
python3 -m http.server 8000
```

Затем откройте:

```text
http://localhost:8000
```

### Через Node.js

```bash
npm start
```

Затем откройте адрес, который покажет `serve`.

## Зависимости

Проект не требует сборки. Визуализация использует CDN:

- KaTeX для отображения формул;
- TensorFlow.js для демонстрационного обучения в браузере.

Для работы нужен доступ к интернету, чтобы браузер мог загрузить CDN-библиотеки.

## Публикация на GitHub Pages

После загрузки репозитория на GitHub:

1. Откройте `Settings -> Pages`.
2. В разделе `Build and deployment` выберите `Deploy from a branch`.
3. Выберите ветку `main` и папку `/root`.
4. Нажмите `Save`.

После публикации страница будет доступна по адресу вида:

```text
https://nurattin.github.io/mlp-forward-animation/
```

## Данные

Файл `mlp_dataset.js` содержит демонстрационный петрофизический датасет и подключается напрямую из `index.html`.
